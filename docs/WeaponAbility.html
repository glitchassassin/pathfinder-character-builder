WeaponAbility

new WeaponAbility(template, updaters, dependencies)

A kind of special weapon enchantment (e.g., "flaming") which may be applied to a weapon.

Parameters:
Name Type Argument Description
template Object <optional>

Template object to copy properties from
updaters Object <optional>

List of custom updater functions to apply to corresponding properties in template object (property names must match)
dependencies Object <optional>

List of arrays of dependency links to establish (property names must match)
